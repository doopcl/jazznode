What is JazzNode?

JazzNode是一款基于Node.js平台的Web服务开发框架，目前已经实现了传统Web Server的基本功能，开发者可以在JazzNode上直接开发面向业务的功能，并且JazzNode 已经支持route特性，所以可以自由的定制站点的URI

最低学习成本

JazzNode使用原生Node.js和少量第三方功能性组件，并未提供深度封装的功能函数，适用于Node.js初学者观摩并快速创建自己的Web应用服务

快速构建服务

JazzNode已实现传统Web Server的基本功能，开发者如没有特殊需求则无需再关注Web Server的实现，可以直接将注意力放在业务的实现上

模板引擎支持

引用最为高效的知名第三方模板art-template组件，开发Server Page毫无压力，PHP,ASP等传统模式web开发者也能轻易hold住，点击查看：artTemplate

移动APP Server利器

最快速编写RESTful API Web Service，JazzNode的route设计天然支持编写通用URI风格的API接口，外加Javascript天然优势，返回JSON格式变得容易

简单快捷自定义

简单易懂的route配置，可以按自己所想定制URI，简易化的访问权限配置，在安全和灵活之间自由选择，没有复杂的调用关系，没有严格的架构约束，只是为了开发者可以快速上手

环境搭建简单

JazzNode所需的环境搭建也十分容易，只需要安装Node.js环境，以及几条npm命令行，就可以完成web服务的搭建，门槛低，任何背景的初学者都能完成

如何开始

环境搭建

点击下载：Nodejs.org 并安装

安装的内容包括Node.js运行环境和npm包管理工具

接下来在终端界面使用npm安装以下组件，全部安装完成之后，环境搭建就完成了


    npm install mime
    npm install mysql
    npm install art-template
    npm install formidable
    npm install cookies

运行服务

Node.js的web服务运行方式为node命令执行包含http.createServer的js代码文件，所以在JazzNode中，运行server_run.js即可运行

    
     
    node github/jazznode/server_run.js

route配置文件

route配置文件是决定对不同请求如何响应的关键配置，JazzNode目前支持二级目录的路由配置


    exports.routes = {
        '':{
            'type':'static',
            'module':{
                'default':'demo/static/index.html',//保留字段
                'index':'demo/static/index.html',
                'favicon.ico':'demo/static/favicon.ico'
            }
        },
        'cgipage':{
            'type':'cgi',
            'module':{
                'default':'demo/modules/cgipage/samples.js',
                'samples':'demo/modules/cgipage/samples.js',
                'samples2':'demo/modules/cgipage/samples2.js'
            }
        },
        'users':{
            'type':'cgi',
            'module':{
                'info':'demo/modules/users/getuserinfo.js',
                'list':'demo/modules/users/getuserlist.js'
            }
        },
        'assets':{
            'type':'static',
            'module':{},
            'restrict':false
        }
    }
